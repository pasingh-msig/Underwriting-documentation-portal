# U360 - Initiative Overview

## Purpose
U360 is the underwriting platform that supports shared services, business-line workflows, and product-specific capabilities across the underwriting portfolio.

## Business Objective
Create a scalable underwriting platform that supports modernization, standardization, improved control, and better integration across underwriting processes.

## Strategic Alignment
U360 is a core modernization initiative and provides the foundation for multi-line underwriting capabilities and shared integrations.

## Status
**Current Status:** In Progress

## Scope
### In Scope
- platform architecture
- shared capabilities
- business-line documentation
- product/sub-line documentation
- audit/control framework

### Out of Scope
- non-underwriting domains
- policy admin replacement details outside integration context

## Initiative Type
- Platform

## Target Operating Role
Shared underwriting platform and system of engagement with integrations to surrounding systems.

## Relationship to Legacy Systems
U360 will coexist with some legacy applications during phased onboarding and migration.
