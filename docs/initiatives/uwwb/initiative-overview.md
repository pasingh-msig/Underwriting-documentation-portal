# UWWB - Initiative Overview

## Purpose
UWWB is the underwriting platform that supports shared services, business-line workflows, and product-specific capabilities across the underwriting portfolio.

## Business Objective
Create a scalable underwriting platform that supports modernization, standardization, improved control, and better integration across underwriting processes.

## Strategic Alignment
UWWB is a core modernization initiative and provides the foundation for shared underwriting capabilities across business lines.

## Status
**Current Status:** In Progress

## Scope
### In Scope
- platform architecture
- shared capabilities
- business-line documentation
- product or sub-line documentation
- audit and evidence framework

### Out of Scope
- non-underwriting domains
- policy administration replacement outside integration context

## Initiative Type
- Platform

## Applications / Components
- shared platform services
- business-line and product-aligned workflows
- common capability integrations
