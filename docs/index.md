# Underwriting Documentation Portal

Welcome to the underwriting documentation portal.

## Purpose
This portal is the shared source of truth for underwriting initiatives, applications, U360 platform documentation, shared capabilities, and audit/control evidence.

## How to use this portal
- Start with **Portfolio Overview**
- Navigate to an initiative for its **Initiative Overview**
- Use **Business View** for business/process content
- Use **Technical View** for architecture, APIs, data, and support
- Use **Audit** pages for evidence traceability and control mapping
