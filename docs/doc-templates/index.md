# Templates

This section contains reusable markdown templates for documenting initiatives, applications, audit evidence, and business lines within the Underwriting Documentation Portal.

## Available Templates

| Template | Description |
|---|---|
| [Initiative Overview](initiative-overview-template.md) | High-level overview of an initiative: goals, scope, stakeholders |
| [Application Business View](application-business-view-template.md) | Business context, capabilities, and value delivered by an application |
| [Application Technical View](application-technical-view-template.md) | Architecture, integrations, and technical design for an application |
| [Operational Controls](operational-controls-template.md) | Operational controls and monitoring for an application |
| [Control Mapping](control-mapping-template.md) | Mapping of controls to audit requirements |
| [Evidence Index](evidence-index-template.md) | Index of audit evidence artefacts |
| [Risk Register](risk-register-template.md) | Risk identification, rating, and mitigation |
| [Decision Log](decision-log-template.md) | Architecture and design decisions |
| [Policy Mapping](policy-mapping-template.md) | Compliance and policy mapping |
| [Cutover Readiness](cutover-readiness-template.md) | Cutover readiness checklist and plan |
| [Benefits Realization](benefits-realization-template.md) | Benefits tracking and realization plan |
| [Audit RACI](audit-raci-template.md) | RACI matrix for audit activities |
| [Legacy Retirement and SOR Mapping](legacy-retirement-and-sor-mapping-template.md) | Legacy system retirement and system-of-record mapping |
| [Business Line Business View](business-line-business-view-template.md) | Business view template for a business line |
| [Business Line Technical View](business-line-technical-view-template.md) | Technical view template for a business line |
| [Product Business View](product-business-view-template.md) | Business view template for a product |
| [Product Technical View](product-technical-view-template.md) | Technical view template for a product |

## How to Use

1. Copy the relevant template file into the appropriate location under `docs/`.
2. Replace placeholder text (marked with `<!-- ... -->` or `[PLACEHOLDER]`) with your content.
3. Add the new file to the `nav:` section of `mkdocs.yml`.
4. Run `mkdocs serve` locally to preview before committing.
